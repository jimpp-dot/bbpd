# external_plugin

这是一个Flutter外接纹理的Plugin。

这个flutter plugin目前只有Android端的实现，使用了SurfaceTexture（共享内存）的方式来实现外接纹理。

介绍如何在flutter端渲染native的OpenGL代码。
